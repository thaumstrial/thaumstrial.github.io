Python爬取静态页面一条龙服务

2023-01-15 Python 爬虫

最近在玩大型语言模型，需要收集互联网上的语料，虽然之前写过这样的代码，但又忘了，特此记录一下。
(￣m￣）

使用fake-useragent库构造随机请求头
由于一些网站在大量请求后会屏蔽爬虫，使用随机的请求头可以解决部分问题。
安装fake-useragent库

pip install fake-useragent

构造随机请求头

from fake_useragent import UserAgent
ua = UserAgent()
headers = {'UserAgent': ua.random}

使用requests库获取页面
安装requests库

pip install requests

使用上文构造的随机请求头获取url页面

import requests
url = '目标网站链接'
req = requests.get(headers=headers, url=url)
html = req.text

使用beautifulsoup4库提取页面中的元素
安装beautifulsoup4库

pip install beautifulsoup4

实例化一个Beautifulsoup

from bs4 import BeautifulSoup
bf = BeautifulSoup(html, 'lxml')

提取指定div标签

# 获取查找到的第一个元素，否则返回None
bf.find('div', class_='指定class')
# 获取查找到的所有元素，否则返回None
bf.find_all('div', class_='指定class')

提取指定p标签中的字符

bf.find('p').string

提取指定a标签中的链接

bf.find('a', id='指定id')["href"]
